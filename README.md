# ArtFetch

This is the AI-based web scraper for **Artverse**, built using Python and Selenium.

## Installation

```bash
git clone https://github.com/Silversoul-07/ArtFetch.git
cd ArtFetch
pip install -r requirements.txt
```

## Usage

```bash
uvicorn app.main:app --reload
```

## Technologies

- Python
- Selenium
